### Project Description

The Support Ticket System is a web-based application developed using Java, Spring Boot, Thymeleaf, Bootstrap, MySQL, and Maven. The system allows users to create support tickets and enables administrators to manage, prioritize, and resolve tickets efficiently.

### Features

* User Login
* Admin Login
* Create Support Ticket
* High and Low Priority Tickets
* Ticket Dashboard
* Add Solution
* Close Ticket
* Delete Ticket
* Ticket Statistics

### Technology Stack

* Java
* Spring Boot
* Thymeleaf
* Bootstrap
* MySQL
* Maven
